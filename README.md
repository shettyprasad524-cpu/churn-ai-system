# 🚀 Customer Churn Intelligence System

## 📌 Problem Statement
Customer churn leads to significant revenue loss. The goal is to predict which customers are likely to churn.

---

## 🧠 Solution
Built an end-to-end machine learning pipeline to predict churn and provide actionable insights.

---

## ⚙️ Features
- Data preprocessing using ColumnTransformer
- Feature engineering (AvgCharges, ServiceCount, SpendIntensity)
- Handling imbalance using SMOTE
- Model comparison (Logistic, RF, HGB, LightGBM)
- Hyperparameter tuning
- Threshold optimization
- Cost-based evaluation
- SHAP explainability
- Streamlit deployment

---

## 📊 Model Performance
- ROC-AUC: ~0.83+
- F1 Score: ~0.62+
